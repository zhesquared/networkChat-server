## Сервер для сетевого чата
___
### Описание:

- Одновременное ожидание новых пользователей и обработка поступающих сообщений со стороны клиента;
- Отправка сообщений клиентам;
- Логирование всех сообщений, проходящих через сервер в файл ```log.txt```;
- Порт для подключения клиентов находится в файле ```settings.properties```;
- Покрыт unit-тестами.
